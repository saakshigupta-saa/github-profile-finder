# GitHub Profile Explorer

A responsive web application built using **HTML, CSS, and JavaScript** that integrates with the **GitHub REST API**. Users can search for GitHub profiles, view repository information, and compare two GitHub users using Battle Mode.

## 🚀 Features

- 🔍 Search any GitHub user
- 👤 Display profile details
  - Avatar
  - Name
  - Bio
  - Company
  - Location
  - Join Date
  - Portfolio Link
- 📂 View the latest 5 repositories
- ⭐ Display repository stars and forks
- ⏳ Loading indicator while fetching data
- ❌ User Not Found (404) error handling
- ⚔️ Battle Mode to compare two GitHub users
- 🌙 Dark Mode
- 📱 Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
- Promise.all()
- GitHub REST API
