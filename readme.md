# GitHub Profile Explorer

A simple web application that uses the GitHub REST API to search GitHub users and display their profile information and latest repositories.

## Features

- Search any GitHub user
- View profile details
- Latest 5 repositories
- Loading indicator
- User Not Found error
- Battle Mode (Compare two users)
- Responsive Design
- Dark Mode

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
- GitHub REST API

## How to Run

1. Download or clone the project.
2. Open the project in VS Code.
3. Run `index.html` using Live Server.
